    在 Windows 11 中，當我們建立新的使用者帳戶時，系統會自動分配一個個人資料資料夾名稱。

    如果你是選擇以 Microsoft 帳戶登入的話，那麼你的使用者個人資料夾名稱通常是根據你的帳戶名稱或電子郵件的前 5 個字元來命名。

    然而，有時候這個名稱可能並不是你想要的。尤其如果你是一個隨便設置電郵的人，那名稱一定會顯得十分精彩......

    那麼，除了在設置時選擇使用本機帳戶登入外，真的別無他法了嗎？ 🤔

    今天博主就和大家聊聊如何安全地更改 Windows 11 中的使用者個人資料資料夾名稱，確保不會影響系統的正常運行 😃

    在開始之前，請留意

    • 你必須具備管理員權限才能執行此操作
    • 更改個人資料資料夾名稱 不會 自動更改 使用者帳戶名稱
    • 此過程涉及修改 Windows 註冊表 (Registry)，請務必謹慎操作，並建議先備份重要資料

    # Step 1：登出目標帳戶

    1. 先登出你想要更改個人資料資料夾名稱的帳戶
    2. 使用其他管理員帳戶登入 Windows 11。如果你沒有其他管理員帳戶，可以像博主一樣，啟用內建的 Administrator 帳戶來執行此操作。

    # Step 3：修改 Windows 註冊表 (Registry)

    1. 開啟「登錄編輯器」(Win + R → 輸入 regedit → 按「確定」(OK))
    2. 前往以下路徑：（將 S-1-5-21-xxxxxxxxxx 替換為你的 SID）
    3. 找到 ProfileImagePath，雙擊打開並修改路徑 e.g. 將 C:\Users\<OLD_NAME> 改為 C:\Users\<NEW_NAME>
    4. 按「確定」(OK)，然後關閉登錄編輯器

    # Step 4：重新命名個人資料資料夾

    1. 開啟檔案總管（Win + E）
    2. 前往 C:\Users ，找到舊的個人資料資料夾 (e.g. <OLD_NAME> )
    3. 右鍵點擊 → 選擇「重新命名」，將其更改為新名稱 (e.g. <NEW_NAME> )
    4. 如果出現「此資料夾正在使用中」(Folder in use . . . file or folder is open in another program) 的錯誤：
      • 請確保目標帳戶已完全登出
      • 重新啟動電腦後再嘗試重新命名

    # Optional 6：（可選）修正 OneDrive 及 Windows 搜尋索引

    1. 如果你使用 OneDrive，請手動變更 OneDrive 資料夾路徑至新的個人資料資料夾名稱
    2. 重建 Windows 搜尋索引，以確保搜尋功能不會指向舊的資料夾名稱

    # 結語

    透過上述步驟，你可以成功更改 Windows 11 使用者個人資料資料夾的名稱，而不會影響系統的穩定性。如果你在過程中遇到錯誤，請確保：

    • 你已完全登出該帳戶
    • 你擁有管理員權限
    • 你已正確修改登錄檔
