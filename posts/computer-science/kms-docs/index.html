    # 支援的產品

    KMS 伺服器可啟用下列 Windows 和 Office 產品:

    • Windows 10 專業版 / 企業版
    • Windows 11 專業版 / 企業版
    • Windows Server 2016/2019/2022
    • Office 2016/2019/2021
    • Microsoft 365 Apps for Enterprise (前身為 Office 365 ProPlus)

    # Windows 啟用步驟

    # 1. 檢查 Windows 版本

    確保您使用的是支援 KMS 啟用的 Windows 版本（專業版、企業版或教育版），家庭版不支援 KMS 啟用。

    # 注意事項

    1. KMS 啟用有效期為 180 天
    2. 系統每 7 天會自動嘗試續期，保持連網即可
    3. 企業網路環境可能會阻擋 KMS 通訊（TCP 1688 端口）
    4. 必須使用 VOL 版本的 Office 才能使用 KMS 啟用

    # 常見問題排除

    若啟用失敗，請嘗試以下步驟：

    1. 確認網路連接是否正常
    2. 檢查防火牆是否阻擋了 TCP 1688 端口
    3. 確認是否使用了支援 KMS 的產品版本
    4. 嘗試使用命令清除註冊表： slmgr /upk 然後重新安裝產品金鑰： slmgr /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

    免責聲明：本服務僅供學習與技術研究使用。請勿用於商業環境，並支持正版軟體。
