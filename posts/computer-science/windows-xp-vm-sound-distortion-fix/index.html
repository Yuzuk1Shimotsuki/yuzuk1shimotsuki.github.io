    這篇文章描述了博主遇到Windows XP虛擬機器音效失真的問題，經過一番嘗試無果後，最終透過修改登錄檔來解決。步驟包括打開登錄編輯程式、新增登錄檔、修改数值等。最後成功解決問題並呼籲讀者重啟電腦應用新設定。文章以簡潔明瞭的方式分享了解決問題的方法，適合需要解決類似問題的讀者參考。

    是這樣的，博主一直以來都在搞虛擬機器，起初還好好的，直到最近電腦壞了重灌後便開始出現問題了。

    就是那台 Windows XP VM 不管怎樣弄，開機音樂還是破破的，伴隨着不少失真......

    找了 Google 大神上很多方法也無解。直到突然想起了以前是透過在主機 (Host) 上改登錄檔解決的，隨即問起了 GPT ，結果一下子就搞定了 🤯🙂‍↕️

    今天就來記錄一下博主當時是如何解決問題的，順便當給自己和大家一些參考

    # Step 1: 開啟登錄編輯程式 (Registry Editor)

    1. 按 Win + R 打開 「執行」 (Run) ，然後輸入 regedit 再按 Enter
    1. 使用系統管理員帳戶驗證登入

    # Step 2: 新增登錄檔 (Registry)

    1. 移至
    1. 右鍵按 New -> DWORD (32-bit) value
    1. 按 F2 重新命名為 GlobalTimerResolutionRequests ，然後按 Enter 進入設定頁面
    1. 修改其 Value data 為 1 並選擇 Base 為 Hexadecimal ，完成後按 Enter 儲存

     

    完成！請關閉登錄編輯程式 (Registry Editor) 並重啟電腦以應用新設定

    倘若一正常的話，恭喜你 🥳🎉！ 你已經成功解決腦人的音效問題了，盡情享受你的 VM 吧！
